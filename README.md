# Simple Node Web Server

## Startup Steps
- Open a terminal and change to the folder containing this server
- Install all dependencies
  - `npm install`
- Set the PORT environment variable
  - `export PORT=8081`
- Start the server
  - `node index.js`

## Testing
- Open a terminal and change to the folder containing this server
- Install all dependencies (if you have not done so)
  - `npm install`
- Run the tests
  - `npm test`
